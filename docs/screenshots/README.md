# Screenshots

Add the following PNG/JPG files to this folder and they'll render in the main
[README](../../README.md) Screenshots section automatically:

| Filename | What to capture |
|---|---|
| `home.png` | The landing page (hero + product grids) |
| `products.png` | The `/products` listing with filters/search |
| `product.png` | A product detail page (gallery, size/colour, reviews) |
| `admin-dashboard.png` | The `/admin` overview (KPIs + charts) |
| `admin-products.png` | The admin products/inventory table |

**Tips for clean shots**
- Use a desktop viewport (~1440px wide) for the storefront and admin.
- Hide the pre-launch gate first (unlock with the passcode, or set `LAUNCH_GATE_OPEN=true` locally).
- Export at 2× / retina if possible for crispness on GitHub.

Once the files are here, the embeds in the README will display them.
